# Design Patterns

This repository contains **implementations of commonly used software design patterns** written with clarity and practical understanding.

The goal of this project is to:
- Understand the **intent** behind each design pattern
- Learn **when and why** to use a pattern (not just how)
- Provide **clean, readable implementations** that can be revisited for interviews and real-world use
